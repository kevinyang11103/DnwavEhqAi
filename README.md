# 外卖点餐系统简介 java1252

## 项目概述

外卖点餐系统是一个集用户、商家、管理员、骑手四位一体的在线订餐平台。系统采用前后端不分离的架构设计，技术栈以 Spring Boot 和 Vue.js 为主。

## 技术栈

- **开发工具**：IDEA、Eclipse
- **编程语言**：Java
- **数据库**：MySQL 5.7+
- **框架**：SSM（Spring、Spring MVC、MyBatis）
- **构建工具**：Maven
